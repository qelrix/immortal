ИНСТРУКЦИЯ ПО ДОБАВЛЕНИЮ МУЗЫКИ
================================

Для того, чтобы музыка "MONTAGEM BOMBO (Slowed) - DJ FallThox" автоматически играла на сайте:

ВАРИАНТ 1 (РЕКОМЕНДУЕТСЯ):
1. Скачайте трек "MONTAGEM BOMBO (Slowed) - DJ FallThox" в формате MP3
2. Переименуйте файл в "music.mp3"
3. Поместите файл music.mp3 в ту же папку, где находятся index.html, style.css и script.js
4. Откройте index.html в браузере - музыка должна играть автоматически

ВАРИАНТ 2:
1. Найдите прямую ссылку на MP3 файл в интернете
2. Откройте файл script.js
3. Найдите строку: audioPlayer.src = 'music.mp3';
4. Замените на: audioPlayer.src = 'ВАША_ССЫЛКА_НА_MP3';

ВАРИАНТ 3 (YouTube):
1. Найдите видео на YouTube с этим треком
2. Используйте онлайн-конвертер для скачивания в MP3
3. Следуйте инструкциям из ВАРИАНТА 1

ПРИМЕЧАНИЕ: 
Современные браузеры могут блокировать автоматическое воспроизведение звука.
Если музыка не играет автоматически, нажмите кнопку воспроизведения (▶).
